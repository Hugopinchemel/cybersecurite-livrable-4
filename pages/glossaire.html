Glossaire Interactif

HTML
HyperText Markup Language, utilisé pour structurer le contenu des pages web.
CSS
Cascading Style Sheets, utilisé pour styliser les pages web.
JavaScript
Un langage de programmation qui permet de rendre les pages web interactives.
Accessibilité
Pratique visant à rendre les sites web utilisables par tous, y compris les personnes handicapées.
